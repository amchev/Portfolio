Thesis Project: Computational Methods for Evaluating FUS Protein Mutation Pathology

The FUS protein or Fused in Sarcoma protein, is an RNA binding protein that regulates various translation and transcription processes. A number of mutations within the FUS protein have been linked clinically to the development of ALS, a debilitating neurdegenerative disease. Following standard evolutionary assumptions, we can extrapolate that less evolutionary variable sites within proteins correspond to amino acid locations critical for normal protein function within the body. This Graduate thesis project focused on creating a statistical model extrapolated from this basic evolutionary principle. By identifying areas of low evolutionary variation, we were able to classify clinically identified mutations as pathogenic or non-pathogenic. Using python data analysis packages and a web based D3 visualization library, this project resulted in both visual and mathematical models for the FUS protein's genetic involvement in ALS on an individual amino acid scale
